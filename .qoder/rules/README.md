# HotClaw + Qoder 协作指南

## 工作流程

1. **打开项目**：在 Qoder 中打开 `D:\project\hotclaw`
2. **查看当前任务**：阅读 `.qoder/rules/current-task.md`
3. **执行开发**：在 Qoder Chat 中说"执行当前任务"
4. **等待完成**：Qoder 会自动读取规则并执行
5. **审核结果**：蟹老板会检查进度并更新任务

## 规则文件说明

| 文件 | 用途 |
|------|------|
| `hotclaw-dev.md` | 项目约束、技术栈、禁止事项 |
| `current-task.md` | 当前具体任务（会持续更新） |

## 如何开始

在 Qoder Chat 中输入：

```
请阅读 .qoder/rules 目录下的规则文件，然后执行 current-task.md 中的任务
```

或者简单说：

```
执行当前任务
```

## 注意事项

- 每次只执行一个任务
- 完成后等待审核
- 不要擅自扩展需求
- 遇到问题及时报告
